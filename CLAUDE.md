# ChatAds TypeScript SDK

Type-safe client for the ChatAds API. Works in Node.js 18+, browsers, and modern edge runtimes.

## Package Info

- **Package name**: `@chat-ads/chatads-sdk`
- **Language**: TypeScript
- **Min Node version**: 18+ (uses built-in fetch)

## Installation

```bash
npm install @chat-ads/chatads-sdk
# or
pnpm add @chat-ads/chatads-sdk
# or
yarn add @chat-ads/chatads-sdk
```

## Usage

See README.md for full usage examples.

## Development

```bash
# Install dependencies
npm install

# Build
npm run build

# Test
npm test

# Lint
npm run lint
```

## Publishing

```bash
# Build and publish to npm
npm run build
npm publish
```

## Project Structure

- `/src` — TypeScript source code
- `/dist` — Compiled JavaScript output (generated)
- `/tests` — Test files
- `/examples` — Usage examples
- `package.json` — Package configuration
- `tsconfig.json` — TypeScript configuration
- `README.md` — User-facing documentation

## Related

- `/api` — Backend API this SDK calls
- `/frontend` — Web UI for managing API keys
- `/sdks/chatads-python-sdk` — Python equivalent
